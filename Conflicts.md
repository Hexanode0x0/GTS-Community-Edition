# Conflicts
This file contains all conflicts and their resolution for GTS CE.
While Vortex lists conflicts both ways, i.e. (A -Conflicts-> B) & (B -Conflicts-> A), the conflicts are only listed one way in this file.
All header mods are from GTS CE.

## Injected Dependencies
Because Vortex can only resolve conflicts with mods that are within the same collecion, GTS CE has to bundle some GTS mods for Vortex to resolve conflicts between CE and GTS.
I've decided to call this Dependancy Injection. CE currently has the following:
- Assorted Mesh Fixes
- Azurite III CS
- Azurite Weathers III
- Durnehviir Redone -Graphics only
- ELFX Shadows
- FuzzBeed's Resources
- GATE TO SOVNGARDE (GTS) - Small Mods and Resources - GTS - Grass
- GATE TO SOVNGARDE (GTS) - Small Mods and Resources - xLod Terrain
- House Cats - My version SE by Xtudo - HeadT
- House Cats- Mihail Monsters and Animals (SE-AE version) (''pet'') - house cats (se-ae) 1.1
- Immersive Spell Learning - DESTified - ISL - DESTified Main File
- Infinite Dragon Variants
- JaySerpa's Small Mods and Resources (Mostly for GTS) - Dyndolod Output
- JaySerpa's Small Mods and Resources (Mostly for GTS) - GTS - Specific Patches
